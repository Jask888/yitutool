# 易图 (Yitu) - Rhino 拆图设计增强工具

易图是一款专为 Rhino 8 深度定制的功能增强工具集，主要服务于工业建模、钣金展开以及数控加工前期的图纸处理。

我们致力于简化繁琐的重复性建模工作，让设计师能将更多精力投入到产品结构本身。

## 主要功能说明

- **高级展开**：针对工业曲面与钣金的展开算法，集成补偿修正。
- **超级偏移**：优化复杂曲线偏移后的交织与断裂问题。
- **图片排孔**：基于位图灰度自动生成冲孔模型或阵列。
- **跟随展开**：支持曲线或物体随曲面自动平推铺展。
- **曲面斜切**：快速生成曲面边缘的 45 度或自定义斜角。
- **平面重建**：智能优化并统一凌乱的平面几何结构。
- **转加工线**：设计图纸一键导出为 CNC/激光切割标准路径。
- **其它工具**：包含夹角度板、包围盒体、曲线打标等 20 余项实用功能。
